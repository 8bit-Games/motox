# X-Moto Mobile

iOS and Android apps using Capacitor.

## Quick Start

```bash
# Install dependencies
npm install

# Build web assets first
cd ../web && npm run build:wasm:release

# Sync to mobile platforms
cd ../mobile
npm run sync

# Open in IDE
npm run open:ios      # Xcode
npm run open:android  # Android Studio
```

## Requirements

- Node.js 18+
- Xcode (for iOS)
- Android Studio (for Android)
- Capacitor CLI

## Platforms

- iOS 13.0+
- Android 5.0+ (API 21+)
